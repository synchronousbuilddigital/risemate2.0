## 🛠️ Development & Deployment Workflow

All source code and active dependencies are managed inside the `parentcompany` subdirectory.

### Initial Setup
To install dependencies, navigate to the active portal directory:
```bash
cd parentcompany
npm install
```

### Dev Server
To start the local developer server with high-velocity Fast Refresh (Turbopack):
```bash
cd parentcompany
npm run dev
```

### Production Build
To validate the builds for production and compile optimal bundles:
```bash
cd parentcompany
npm run build
```
